Data Groups


Individual user accounts can be tagged with one or more data groups through the profile API.

These tags allow to process the data from participants. For example, you may wish to define a "test_users" data group and assign testers to this group, so you can filter out their data when analyzing results.

You can also use these tags to filter scheduled activities for groups of users, generating different data for different study participants.

These tags cannot include identifying information about specific users. They are exported with the study data set. If you need to identify users, please use the external identifier API.
