User Attributes


Bridge stores personally identifiable information separately from the health information collected for a study. By default, we accept a participant’s first and last name, and an email address, through that participant’s user profile.

You can configure your study to accept additional information about a participant. These fields can then be added to a user profile and they will be saved and included as part of the Participant Roster.

Like a participant’s name or email address, this information will not be included in the data set exported from the study.
