Text Message Templates

When formatted, the message will be too long for SMS (max. 160 characters).

A study with a workflow that consents a user prior to registering for an account, can direct the user to an app store to download and install an app. This is the message that is sent to the user with the link.

It must contain the value ${appInstallUrl}. This will be the link to the app in the app store.

There are other values available for use in your template:

${studyShortName}
the short name you entered for your study
${sponsorName}
the name you entered for the group sponsoring the study
${supportEmail}
your support email address
${expirationPeriod}
the period before authentication link/token in the message will expire (e.g. "2 hours")
