Scheduling Events

Schedule for
after enrollment

At the time the participant enrolls in a study, you can create additional events for the user (beyond the time the user enrolled) against which you can schedule activities.

For example, if you wanted to have four periods over a year when the user does a sequence of activities for a few days, you could create event keys for each of these periods, and then schedule sequences against those events. The events could be at 3, 6, and 9 months after enrollment. These events would all be created for the user when they enroll and will be visible in the users event map.

When creating a schedule, you will be able to select that event key in the "when to start schedule" editor.
