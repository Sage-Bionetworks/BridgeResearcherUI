Verify Email Reset Password Email Sign In

This is the email sent to a user that will contain a link to sign in to the app. This requires coordination with the app itself to capture the link and process the token accordingly.

It must contain the value ${token}, which is a value the app must parse out of the URL and return to the server to complete sign in.

You will also need to change the URL to use a custom host. If you used the default ${host} value it will conflict with other apps built on the Bridge server. The initial ${host} value is for testing purposes.

There are other values available:

${studyName}
the name you entered for your study
${sponsorName}
the name you entered for the group sponsoring the study
${supportEmail}
your support email address
