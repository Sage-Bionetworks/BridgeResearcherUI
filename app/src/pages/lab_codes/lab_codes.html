Lab Code

Lab codes allow you to register devices using a code, in a controlled setting like a lab.

Import the code here and an app can use it to authenticate with the Bridge server (if it has been designed to do so, see the help documentation).

Bridge APIs are public and this code will be used to create credentials that authenticate the device, so it should be as complex and hard to guess as a good password.

These codes will be exported with your data to Synapse as a field named externalId.
