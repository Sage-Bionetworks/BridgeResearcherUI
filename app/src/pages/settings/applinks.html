Apple Universal Links

Android App Links

To create links that are intercepted by your application (“App Links” on Android, “Universal Links” on iPhone), we provide the server support to register those links with your application.

Your links must point to the host and path https://webservices.sagebridge.org/<your-study-id>. You should define paths here for Apple OS that always include this study-based context path.

On Android, you should declare the Intents to capture these links so they include the <data … android:pathPrefix="/<your-study-id>"/> element to scope your captured URLs.

Bridge enforces these constraints.
