Assessments

A published assessment can be imported into an app and customized through our customization API. Here you can define one or more “property inspectors” that will indicate which fields a study designer can change without invalidating the shared assessment.

This assessment is not derived from a shared assessment, so you can edit the entire configuration JSON right now (in a separate tab). As an assessment author, you are responsible for defining the configuration, as well as the fields that can be configured without invalidating the assessment (if any).
