Task Identifiers



You can schedule activities for participants that include either surveys or tasks.

Surveys can be created here through the research manager, then scheduled to run in your application through the Bridge SDK.

Tasks are known only to your application. However you can schedule them by assigning them unique identifiers. A complete list of these should be defined here.

Once this is done, these identifiers will be available to you when you schedule tasks in the scheduler. Your application should know what task to run when a schedule references one of these identifiers.
