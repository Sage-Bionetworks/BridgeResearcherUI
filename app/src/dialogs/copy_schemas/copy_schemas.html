Copy upload schemas

To copy this schema, choose a new ID for your copy. You can also change the name if you wish.

NOTE: New schema revisions will be exported to a new Synapse table. Many schema changes can be made to an existing schema and they will not require a new table. Give it a try—the editor will stop and tell you if it needs to change the revision number.

You can also set minimum and maximum application versions on specific fields, and those fields will only apply to those versions of your application. This also prevents your data from going into a new Synapse table. See subsequent app releases for more details.
